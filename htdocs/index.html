C Web Server

A Simple and Fast Web Server programmed in C

Features

  • Supports simple GET requests (without parameters).
  • Supports multiple file types:
    • HTML
    • CSS
    • JavaScript
    • JPEG
    • PNG

Limitations

  • No support for GET parameters and other types of request methods.
  • No caching policy.
  • Not every file type is supported.
  • Not suitable for production

Getting Started

Navigate to the htdocs directory and add your files. Then, build and run the server executable by executing make run .
